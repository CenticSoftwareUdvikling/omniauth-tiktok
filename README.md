# Omniauth::Tiktok

Omniauth Strategy for Tiktok

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'omniauth-tiktok', git: 'https://github.com/deepdivr/omniauth-tiktok'
```

And then execute:

    $ bundle install
