# Omniauth::Tiktok

Omniauth Strategy for Tiktok

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'omniauth-tiktok'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install omniauth-tiktok
